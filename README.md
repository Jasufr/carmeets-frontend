# Front-end for Car Meetings App 🏎️

React NextJS Front-end App listing Car Meetings around Tokyo.
This app fetch data from this API [Rails API for Car Meetings App](https://github.com/Jasufr/carmeets-api).

<br>

### Getting Started
### Setup
```
npm install
```

### Run a server
```
npm run dev
```
